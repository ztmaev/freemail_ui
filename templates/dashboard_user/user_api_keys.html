API Usage

How to use your API key to send emails

HTTP Request

Make a POST request to our endpoint with your API key and email details: 

POST {{ email_api_url }}
Content-Type: application/json

{
  "api_key": "your_api_key",
  "sender_name": "Your Name",
  "subject": "Email Subject",
  "message": "Your email content here",
  "message_type": "plain", or "html"
  "receiver_email": "recipient@example.com",
  "footer": "Optional footer text" optional
}

Response

You'll receive a JSON response with the status of your request: 

{
"status": "success",
"message": "Email sent successfully",
"email_id": "email_12345"
}
